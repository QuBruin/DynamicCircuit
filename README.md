# DynamicCircuit
This is the repository of 2024 Yale quantum hackthon.




# Initialize a virtual environment


## Windows


```console
cd path\to\your\project
```

```console
py -m venv yalehack
```

```console
yalehack\Scripts\activate
```

```console
pip install -r requirements.txt
```


## Linux/Mac

```console
cd path/to/your/project
```

```console
python3 -m venv venv
```

```console
source venv/bin/activate
```

```console
pip install -r requirements.txt
```
